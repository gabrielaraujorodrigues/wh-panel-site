# WH-Panel — WhatsApp Bot Hosting Panel

Painel web para hospedar e gerenciar bots de WhatsApp 24/7.

## Stack
- React + Vite + Tailwind CSS (frontend)
- Express 5 + WebSocket (API)
- PostgreSQL + Drizzle ORM
